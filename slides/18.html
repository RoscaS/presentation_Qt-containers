### Itérateurs Java style
* `QListIterator`: lecture seule ```cpp QList strings{"un", "deux", "trois"}; QListIterator i(strings); while (i.hasNext()){ qDebug() << i.next(); } // équivalent à: foreach (QString str, strings) qDebug() << str; ```
* `QMutableListIterator` lecture et écriture ```cpp QList lst{1, 2, 3}; QMutableListIterator i(int); while (i.hasNext()) { i.next() i.setValue(0) } ```
