### Itérateurs STL
* Compatible avec les algorithmes génériques de la STL * **Vitesse d'exécution optimale**
* Un pour la lecture et l'écriture ```cpp QList strings {"UN", "DEUX", "TROIS"}; QList::iterator i; for (i = strings.begin(); i != strings.end(); ++i) { *i = (*i).toLower(); } ``` * Un pour la lecture seule ```cpp QList::const_iterator i; for (i = strings.begin(); i != strings.end(); ++i) { qDebug() << *i; } ```
